# Timer
A simple countdown timer.

## Functions
- Calculate duration based on start time.
- Play alarm when the timer ends.
- Set/Unset window always on top.
- Prevent screen off while the timer is running.
- Smart typing mode time editor.
  - Default mode is overtype.
